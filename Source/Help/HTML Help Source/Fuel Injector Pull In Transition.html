Pull In Transition Settings

The analysis needs to find the three sections of a stage (pull in, hold in, and off). There are multiple configurable approaches for finding these sections. This setting helps define the transition from pull in to hold in.

Pull In Transition Settings

Option 1: Pull In Transition Dwells uses the number of found voltage pulses to determine when the pull in section is done. The channel value sets the number of pulses to look for.

Option 2: Pull In Transition Current Drop measures the current drop after finding a max value to determine when the pull in section is done. The channel value sets the drop level.

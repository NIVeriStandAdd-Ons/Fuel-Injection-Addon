Global Settings

These are settings that apply to all injectors.

Settings

Enable Injection Capture: Enables the monitoring and processing of injection events.

Initial Voltage Threshold: The value used as the trigger level for finding the start of an injection event.

Injection Capture Duration: The acquisition duration after finding the injection event. This value should be larger than the longest expected injection event.

Voltage Gain: Applies a gain or scaling to the voltage signal on the FPGA.

Current Gain: Applies a gain or scaling to the current signal on the FPGA.

Filter Duration: The time that the voltage channel must be above the Initial Voltage Threshold to be considered the start of an injection event. This helps filter out any spurious noise on the voltage channel.

Pre Trigger Samples: The number of samples to include in an injection event waveform that happened before the event trigger.
